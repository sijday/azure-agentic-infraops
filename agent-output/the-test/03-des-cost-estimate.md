## ✅ Decision Summary

| Decision | Cost Impact | Rationale |
| --- | --- | --- |
| AKS over ACA | +$438/mo management fee (AKS Standard) | PCI CDE isolation requires dedicated node pools. ACA would save ~$438/mo but lacks node-level isolation. |

| Azure Firewall Premium | +$1,278/mo fixed | IDPS capability for PCI-DSS Req 1. Standard tier lacks IDPS — non-compliant for PCI. |
| Key Vault Premium (HSM) | +$22/mo over Standard | PCI-DSS requires HSM for cryptographic keys. Minimal cost delta. |
| PostgreSQL Memory Optimized | +$500/mo over General Purpose | Write-heavy payment transactions benefit from larger memory. Can start with General Purpose ($1,092/mo) to save. |

## Architecture Overview

```
Internet → Azure Front Door Premium (WAF + DDoS)
              ↓
         Azure Firewall Premium (IDPS)
              ↓
         API Management Premium (VNet)
              ↓
         AKS (D8s_v5 × 3-20 nodes)
           ├── Payment Service → PostgreSQL Flex Server (E16s v5)
           ├── Session Service → Cosmos DB (Autoscale)
           ├── Queue Worker → Service Bus Premium
           └── All Services → Key Vault Premium (HSM)
              ↓
         Log Analytics (1-year retention)
         Microsoft Defender for Cloud
```

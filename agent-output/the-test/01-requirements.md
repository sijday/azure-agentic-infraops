## Summary for Architecture Assessment

### Recommended Architecture

| Component | Recommendation | Justification |
| --- | --- | --- |
| **Compute** | **Azure Kubernetes Service (AKS)** over Azure Container Apps | AKS provides: dedicated node pools for CDE isolation, Kubernetes network policies for micro-segmentation, KEDA for event-driven autoscaling, mature PCI-DSS compliance tooling (Falco, OPA Gatekeeper), and fine-grained control needed at 10k TPS. ACA lacks dedicated node isolation and granular network policy required by PCI-DSS. |
| **Database (Primary)** | Azure Database for PostgreSQL Flexible Server | ACID transactions for payment records, zone-redundant HA, read replicas for reporting |
| **Database (Cache/State)** | Azure Cosmos DB (NoSQL API) | Sub-10ms latency for session state, distributed caching, autoscale RU/s |
| **API Gateway** | Azure Front Door Premium + Azure API Management (Premium, VNet-integrated) | WAF, DDoS, global load balancing, API rate limiting, OAuth, VNet integration for CDE |
| **Networking** | Hub-spoke VNet with Azure Firewall Premium | CDE isolation, IDPS, east-west traffic inspection |
| **Messaging** | Azure Service Bus Premium | Dedicated capacity for transaction queues, dead-letter handling |
| **Secrets** | Azure Key Vault Premium (HSM) | PCI-DSS cryptographic key management |
| **Monitoring** | Azure Monitor + Log Analytics + Defender for Cloud | Full observability stack with 1-year PCI log retention |
| **DDoS Protection** | Azure DDoS Network Protection (Standard) | Full VNet coverage for CDE network |
| **Container Registry** | Azure Container Registry Premium | Geo-replicated, vulnerability scanning, content trust |
| **Identity** | Entra ID + Workload Identity + PIM | Zero-trust, managed identity everywhere, JIT admin access |

### Key Architecture Decisions for Architect Agent

1. **AKS over ACA**: PCI-DSS CDE isolation requires dedicated node pools and Kubernetes network policies not available in ACA
2. **Dual Database**: PostgreSQL for ACID transactions + Cosmos DB for low-latency cache/session — reduces load on primary DB
3. **Hub-Spoke Network**: Azure Firewall Premium provides IDPS and east-west inspection required by PCI-DSS Req 1
4. **Azure Front Door Premium**: WAF + DDoS + global anycast required for internet-facing payment endpoint
5. **Key Vault Premium (HSM)**: PCI-DSS requires hardware security modules for cryptographic key storage
